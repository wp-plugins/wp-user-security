== Description ==

This plugin enables you to reset a users password (or multiple users passwords) by interval. This makes your WordPress site much safer for brute force attacks, because you can reset your password daily by a 130 character random password. 

I hear your think "but if I automatically reset my password everyday, how do I know it?". Well, you don't :) But you don't have to. Via your [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230) dashboard you can automatically log in to your WordPress site via the connection/key of [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230). So you don't need your password at all.

If you don't have a [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230) account yet you can set it up for free (few sites). If you don't use [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230) and you don't want to use it in the future as well, this plugin isn't one you gonna need. 

= Nederlands =
Met de WP User Security plugin heb je de mogelijkheid om voor je eigen gebruiker of alle gebruikers uit je WordPress site het wachtwoord automatisch te resetten naar een 130 karakters tellend wachtwoord. 

Omdat je hierdoor beschikt over een dagelijks / wekelijks of maandelijks gereset wachtwoord van 130 karakters ben je optimaal beveiligd tegen brute force attacks.

Het wachtwoord dat wordt gereset weet je zelf uiteraard niet. Hoe log je dan in zul je denken? Niet. Omdat de plugin een uitbreiding is op de diensten van [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230) hoef je niet meer in te loggen via het inlog scherm. Omdat je inlogd vanuit het dashboard van [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230) heb je zelf je gebruikersnaam en wachtwoord niet meer nodig. 

Heb je nog geen [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230) account dan kun je die gratis aanmaken. Tot een aantal sites blijft de dienst gratis. Heb je geen account en wil je ook geen gebruik gaan maken van [ManageWP](https://managewp.com/?utm_source=A&utm_medium=Link&utm_campaign=A&utm_mrl=2230)? Dan is dit niet een plugin waar je wat aan gaat hebben.

== Installation ==
1. Upload the plugin to your WordPress site. This can be done manually (through FTP) and through the WordPress dashboard (as *.zip file)
2. Activate the plugin through the plugin page
3. Set the interval through Security -> settings and select the user for whom the password must be reset
